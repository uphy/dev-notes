# テキストエディタ: Emacs

$ brew install emacs

--with-cocoa指定するとターミナル外でも表示できるが、GUI環境でまで使わなくてよいので指定しなかった。

# スクリーンショット: Kap

$ brew cask install kap

# パスワード管理: KeePass

  1. https://www.keepassx.org/ からをダウンロード
  2. ダウンロードした*.dmgファイルを開いて、KeePass.appをApplicationsにコピー
